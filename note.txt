# Cấu trúc folder dự án

index.html --> file chạy chính

- scss --> chứa các file scss
  - style.scss --> file chạy chính scss (Nối các file khác)
  * partials
    _base.scss
    _mixin.scss
    _reset.scss
    _variables.scss
  * components
    _header.scss
    _footer.scss
    _breadcrums.scss
  *pages
    
- images --> chứa các file ảnh
- fonts --> chứa các file fonts
- css --> chứa các file css được generate từ scss
- pages --> các file html tương ứng với các trang
